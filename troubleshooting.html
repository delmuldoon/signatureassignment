Common Troubleshooting

Before you call in, see if this helps...

Troubleshooting Topics

  • Hardware Issues
  • Software Issues
  • Network Issues

Hardware Issues

  • Computer won’t turn on or randomly shuts down
  • Battery not charging or draining quickly
  • Screen is black, flickering, or damaged
  • Keyboard or trackpad not responding
  • USB devices or peripherals not being detected

Software Issues

  • Apps or programs crashing unexpectedly
  • Operating system update failures
  • Computer running slow or freezing
  • Files or programs not opening correctly
  • Pop-ups or suspected malware activity

Network Issues

  • Wi-Fi not connecting or constantly dropping
  • Slow internet speeds
  • No internet access despite being connected
  • Bluetooth devices failing to pair
  • VPN or network configuration issues
